
## Analyzer App

### See it in action

1. Clone this repo!
1. ```python -m SimpleHTTPServer```
2. head to [http://localhost:8000](http://localhost:8000)

### Get a development environment up and running

1. ```npm install```
2. ```npm start```
3. head to [http://localhost:8080](http://localhost:8080)

### Build for production

```npm run build```
